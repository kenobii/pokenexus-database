# PokeNexus Database

Base de dados completa do jogo **PokeNexus** (poke-nexus.com), anteriormente conhecido como Pokemon Planet.

## Conteúdo

- `pokenexus_database.json` — Banco de dados principal com todas as mecânicas, localizações, NPCs, itens, lendários, sistemas e guias do jogo.

## Atualização

Este banco de dados é atualizado automaticamente toda semana por um agente de IA que busca novidades, patches e atualizações do jogo nas principais fontes:
- [PPOBuddy](https://www.ppobuddy.com)
- [Pokemon Planet Fandom Wiki](https://pokemon-planet.fandom.com)
- YouTube (canais oficiais do PokeNexus)
- Site oficial: [poke-nexus.com](https://poke-nexus.com)

## Versão atual

Veja `database_version` dentro do JSON.
